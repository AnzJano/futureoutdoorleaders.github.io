# Future Outdoor Leaders

Future Outdoor Leaders is a nonprofit focused on helping young people build confidence, leadership, and life skills through outdoor experiences.

## Website
This site is hosted with GitHub Pages.

## How to update the site
- Edit `index.html` to change the page content.
- Edit `styles.css` to change colors, fonts, and layout.
- Commit changes to publish updates.
